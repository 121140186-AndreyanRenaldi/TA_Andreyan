# Estimasi Kecepatan Kendaraan Menggunakan Optical Flow
## Overview
Proyek ini menggunakan algoritma optical flow untuk mengestimasi kecepatan kendaraan, dengan memanfaatkan pergerakan piksel antar frame pada video.
